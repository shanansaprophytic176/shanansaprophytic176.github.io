Markdown 是一种轻量级标记语言，让你可以用纯文本格式编写文档。以下是常用语法总结。

## 标题

用 `#` 号表示标题，一个 `#` 是一级标题，最多六个。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 文本样式

```markdown
**粗体文本**
*斜体文本*
***粗斜体文本***
~~删除线文本~~
`行内代码`
```

效果：**粗体文本**、*斜体文本*、***粗斜体文本***、~~删除线文本~~、`行内代码`

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 子项目
```

效果：
- 项目一
- 项目二
  - 子项目

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

效果：
1. 第一步
2. 第二步
3. 第三步

## 链接与图片

```markdown
[链接文字](https://example.com)
![图片描述](图片URL)
```

## 引用

```markdown
> 这是一段引用文字
> 可以多行
```

效果：
> 这是一段引用文字
> 可以多行

## 代码块

用三个反引号包裹代码，可以指定语言实现高亮：

````markdown
```python
def hello():
    print("Hello, World!")
```
````

效果：

```python
def hello():
    print("Hello, World!")
```

## 表格

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:-------:|-------:|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

效果：

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:-------:|-------:|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

## 分割线

用三个或更多 `-`、`*` 或 `_`：

```markdown
---
```

## 任务列表

```markdown
- [x] 已完成
- [ ] 未完成
- [ ] 待办事项
```

效果：
- [x] 已完成
- [ ] 未完成
- [ ] 待办事项

## 脚注

```markdown
这里有个脚注[^1]

[^1]: 这是脚注内容
```

## 数学公式

行内公式用 `$...$`，块级公式用 `$$...$$`：

```markdown
$E = mc^2$

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$
```

## 快捷键速查

| 功能 | 语法 |
|------|------|
| 加粗 | `**文本**` |
| 斜体 | `*文本*` |
| 标题 | `# 标题` |
| 链接 | `[文字](链接)` |
| 图片 | `![描述](URL)` |
| 代码 | `` `代码` `` |
| 引用 | `> 引用` |
| 列表 | `- 项目` |
