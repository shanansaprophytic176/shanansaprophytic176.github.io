## 研究背景

随着现代生活节奏的加快，许多家庭主人因为工作需要经常出差，导致家中的绿植长时间无人照料。植物因长期缺乏水分和光照，容易出现叶子枯黄、生长不良等问题，严重时甚至会导致植株死亡。

为了解决这一问题，本项目设计了一套**家用智能灌溉系统**，能够自动监测环境数据并智能控制灌溉，让用户即使不在家也能呵护心爱的绿植。

## 系统概述

本项目旨在构建一套适合家庭使用的智能灌溉系统，主要功能包括：

- **实时监测**：采集土壤湿度、温度、光照等环境数据
- **智能灌溉**：根据监测数据自动调节灌溉水量和时间
- **语音控制**：集成语音模块，支持语音指令操作
- **远程操控**：通过WiFi将数据上云，支持手机小程序远程控制

## 系统架构

智能灌溉系统采用三层架构设计：

```
┌─────────────────────────────────────────────────────┐
│                   应用层                             │
│   串口屏显示 │ 小程序控制 │ 云平台数据存储          │
├─────────────────────────────────────────────────────┤
│                   网络层                             │
│        ESP8266 WiFi │ ZX-D30 蓝牙                   │
├─────────────────────────────────────────────────────┤
│                   感知层                             │
│   土壤湿度传感器 │ 温度传感器 │ 光照传感器          │
└─────────────────────────────────────────────────────┘
```

| 层级 | 组件 | 功能 |
|:----:|:-----|:-----|
| **感知层** | 土壤湿度传感器、DS18B20温度传感器、BH1750光照传感器 | 采集环境数据 |
| **网络层** | ESP8266 WiFi、ZX-D30蓝牙 | 数据传输与通信 |
| **应用层** | TJC3224T124串口屏、继电器水泵、云平台小程序 | 用户交互与控制 |

## 硬件设备

系统核心硬件包括：

| 设备 | 型号 | 功能说明 |
|:-----|:-----|:---------|
| 主控芯片 | **STM32F103C8T6** | 系统核心，负责数据处理和设备通信 |
| 温湿度传感器 | DHT11 | 实时采集环境温湿度 |
| 光照传感器 | BH1750 | 监测光照强度 |
| 土壤湿度传感器 | 电容式 | 检测土壤含水量 |
| 语音模块 | ASRPRO | 实现语音控制功能 |
| WiFi模块 | ESP8266 | 无线网络连接，数据上云 |
| 蓝牙模块 | ZX-D30 | 近距离无线通信 |
| 串口屏 | TJC3224T124 | 显示系统状态 |
| 继电器模块 | 5V继电器 | 控制水泵开关 |
| 水泵 | 直流潜水泵 | 执行灌溉操作 |

## 软件设计

系统软件基于 MQTT 协议实现数据上云，用户可通过小程序远程监控和控制灌溉系统。

### 数据流程

```
传感器采集 → STM32处理 → ESP8266上传 → 云平台存储 → 小程序显示
                                    ↓
                              语音模块本地控制
```

### 主要功能模块

1. **数据采集模块**：定时读取各传感器数据
2. **灌溉决策算法**：根据土壤湿度自动判断是否灌溉
3. **通信模块**：通过MQTT协议与云平台通信
4. **用户界面**：串口屏本地显示 + 小程序远程控制

## 总结与展望

本项目实现了家用智能灌溉系统的基本功能，能够自动监测环境数据并智能控制灌溉。未来可从以下方面继续优化：

- **AI 算法**：引入机器学习，根据植物种类和生长阶段优化灌溉策略
- **多传感器融合**：增加雨滴传感器、水位传感器，实现更精准的控制
- **能源管理**：集成太阳能供电，实现绿色环保
- **语音交互**：升级语音模块，支持更自然的对话交互
